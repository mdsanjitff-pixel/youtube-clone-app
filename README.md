# YouTube Clone App

A simple YouTube-style Android app built with **React Native + Expo**.

## Features (Planned)
- Home screen with video list
- Video player with full controls
- Search functionality
- Dark mode
- Local video upload
- Bottom navigation (Home, Shorts, Library, etc.)
- Smooth UI like original YouTube

## Technologies
- React Native (Expo)
- React Navigation
- `react-native-video` for playback
- Axios for future API calls
- Tailwind / NativeWind (for styling)
